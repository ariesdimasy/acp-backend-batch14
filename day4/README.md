## Array , Slice, and Map

cara membuat array

```go

// array yang sudah di tentukan jumlah dan isi nya

var colors = [3]string{"red", "yellow","brown"}
```

cara membuat array multi dimensi

```go
multi := [][]int{} // ini slice
multi = append(multi, []int{1, 2})
multi = append(multi, []int{3, 4})

// [[1 2] [3 4]]
```
